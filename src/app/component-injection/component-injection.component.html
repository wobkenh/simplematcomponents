Custom Components

Simplemattable offers quite powerful tools to customize the way table cells appear. However, sometimes, this might now be enough. In such cases, you can tell simplemattable to render one of your own components into the table cells.

If your component has input parameters that need to be filled with data from your model, you can specify a function in which you can acces the component instance and the data. In this function, you can fill in the input parameters of your component yourself.
