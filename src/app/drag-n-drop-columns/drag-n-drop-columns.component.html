Reordering columns by drag and drop

Allows the user to reorder the columns by dragging the table headers around. Activate this by setting the 
columnDragAndDrop
flag.
visibility Display
code Code

Typescript - Data

this.dataSimple = [
  new ComplexTestData(1, 40, 'test1', null, 'test2'),
  new ComplexTestData(2, 42, 'test2', null, 'test3')
];

Typescript - Columns

this.columnsSimple = [
  new TableColumn<ComplexTestData, 'id'>('My ID', 'id'),
  new TableColumn<ComplexTestData, 'value'>('My Value', 'value')
];

HTML

<smc-simplemattable [data]="dataSimple" [columns]="columnsSimple"
                    [columnDragAndDrop]="true"></smc-simplemattable>
