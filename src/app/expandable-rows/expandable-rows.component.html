Expandable Rows

Want to show the user some details when he clicks on a row? Then define a component for your row detail content that implements the DetailRowComponent interface. You can then pass the type to simplemattable's detailRowComponent input parameter. The data (which element was clicked) will be passed into the Detail Component via the onInput method, which is defined in the DetailRowComponent interface.

Note that the detail components are not generated when the table is first displayed, but only when the corresponding row is opened.
