Scrollable Tables

If you want the content of simplemattable to be scrollable, set "overflowAuto" to true and limit the height of the simplemattable element. If you want to keep the header row at the top even when scrolling, use the sticky attribute. When using sticky, overflowAuto is automatically implied.
